# Walljump-gamemode
Open source walljump gamemode.


The server config should look something like this.

    sv_accelerate 0
    sv_friction 0
    sv_airaccelerate 0
    sv_maxrate 0
    sv_minrate 100000
    sv_maxupdaterate 99
    sv_maxcmdrate 99

To change the tickrate to 100 you add:

    -tickrate 100
   
In launch options
